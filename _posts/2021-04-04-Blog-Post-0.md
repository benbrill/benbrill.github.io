# Creating the Visualization

So, there are quite a lot of different variables to choose from. Some are quantitative and others are qualitative. 

What we do know is that the penguins' bodily measurements (like Culmen Length, Culmen Depth, etc.) are key to classifying a penguins species. So we will graph the relationship between culmen length and depth between different species.

For this, we will use the `seaborn` library, which makes it a lot easier to interface with `matplotlib`


```python
import seaborn as sns
sns.set_theme()
```




```python
mapDict = {'Adelie Penguin (Pygoscelis adeliae)': "Adelie",
 'Chinstrap penguin (Pygoscelis antarctica)': "Chinstrap",
"Gentoo penguin (Pygoscelis papua)": "Gentoo" }
penguins.Species = penguins.Species.map(mapDict)
penguins
```

We will compare the relationship between Culmen Length and Culmen Depth across the three species of Penguins measured. For this, we will use a jointplot, which will graph a point for each penguin corresponding to its Culmen Length and Depth, along with providing the distribution of each variable along the axis, so we can better compare the differences between species. We will differentiate the species by passing a `hue` argument to change the color of each point correspong to the species


```python
sns.jointplot(x = "Culmen Length (mm)",
              y = "Culmen Depth (mm)", 
              data = penguins, 
              hue = "Species", )
```




    <seaborn.axisgrid.JointGrid at 0x1a404c6dd90>

And our visualization is complete! We can see that most penguins of the same species have a similar Culmen Length and Depth. We also can tell by the density plots that Chinstrap Penguins have a wider range of potential Culmen Lengths and Depths than Adelie and Gentoo Penguins
